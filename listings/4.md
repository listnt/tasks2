Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {

	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
		//close(ch)
	}()
	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа по очереди печатает значения из канала, после чего вылетает ошибка. 
Решение: необходимо закрыть канал

```
